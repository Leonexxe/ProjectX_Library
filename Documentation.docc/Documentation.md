# ``projectX Library``

#### the [projectX Library](./Documentation.md) is a general purpose library for c++.
#### It comes as source code that can be put into the include folder to provide a plug and play experience.

## ``Overview``

#### The ProjectX library includes classes and functions made to complete mathematical and geometric operations or manipulate text in a variety of ways. There is also functionality that allows to do network operations like sending or receiving network messages with ease. Further more this Library provides a portable way to display graphical user interfaces directly from c++ without needing extensive knowledge on [openGL](https://www.opengl.org) or [Metal](https://developer.apple.com/documentation/metal)
#### The projectX library has a lot of options for customization, which include but are not limited to the following options
- Private members of classes can be made public on a per class basis
- instead of using straight up hard coded types the library uses types such as ``__int__`` or ``__string__`` which can be defined via macros
- the library uses it's own values for constants such as Ï€ which is set as 3.1415, these values can be changed via macros
